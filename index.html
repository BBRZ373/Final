How to play

  • Click the roll dice button to start the game.
  • Click the reset game button to reset the game.
  • Notice! ! ! The game will only be reset after clicking the reset game button, and other operations will be invalid.

Game Rules

  • Each player rolls two dice per turn. The following rules apply:
  • If either dice shows a 1, the player scores 0 for the round.
  • If both dice show the same number, the round score is doubled.
  • Otherwise, the round score is the sum of the two dice.
  • Whoever gets the higher score wins
